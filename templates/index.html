API Achô

Uma API moderna e poderosa para categorização versão 1.0.1.

Visão Geral

Essa API foi desenvolvida para processar e recuperar métricas de similaridade textual de forma eficiente, utilizando aprendizado de máquina e embeddings. Além disso, ela é capaz de integrar-se a modelos generativos avançados, como o Gemini, mas essa integração só ocorre quando o nosso modelo proprietário não está confiante nos resultados. Isso garante economia de custos e tempo, já que o Gemini possui um tempo de resposta extremamente alto.

Nosso modelo interno foi treinado com mais de 12 mil pares de dados, garantindo um desempenho consistente e eficaz em diversos cenários. Ele está em constante evolução, sendo atualizado regularmente com novos dados para aumentar sua precisão e abrangência.

Benefícios da integração com o Gemini:

  • Garante alta precisão em cenários mais complexos.
  • Reduz a probabilidade de erros ao lidar com textos ambíguos ou pouco claros.
  • Oferece uma abordagem híbrida, combinando velocidade do modelo interno e a sofisticação do modelo generativo quando necessário.

Sugestões de Categorias

Aqui você pode enviar sugestões de novas categorias para a API. Para que a sugestão seja aprovada, você deve informar um token válido.

Como Funciona

Quando você submete uma nova categoria, a API irá checar o seu token. Caso seja válido, a sugestão segue para análise e, se aprovada, entrará para a lista de categorias. Caso contrário, a sugestão será rejeitada.

Exemplo de Requisição

POST /suggest_category
Content-Type: application/json

{
    "suggestion": "Itens para Camping e Lazer ao Ar Livre",
    "token": "seu-token-aqui"
}

Exemplo de Resposta

{
    "status": "aprovado",
    "mensagem": "Sua sugestão foi encaminhada para análise."
}
